# Анализ периодичности импульсного сигнала

Небольшой учебный проект на Python для генерации и анализа периодического сигнала, состоящего из пачек импульсов.

Проект демонстрирует, как можно определить период повторения сигналов с помощью методов обработки сигналов.

---

## Возможности

Проект позволяет:

- генерировать периодические пачки импульсов
- добавлять случайные смещения времени импульсов (jitter)
- находить период сигнала
- анализировать сигнал в **offline** и **online** режиме
- строить графики сигнала и результатов анализа

---

## Структура проекта

project/
│
├── main.py
├── generator.py
├── analyzer.py
└── visualization.py


**generator.py**

Генерирует тестовые сигналы из повторяющихся пачек импульсов.

**analyzer.py**

Ищет период сигнала. Поддерживает:

- offline анализ (весь сигнал сразу)
- online анализ (поток данных)

Методы поиска периода:

- автокорреляция
- FFT (преобразование Фурье)

**visualization.py**

Отвечает за построение графиков:

- сигнал
- автокорреляция
- спектр FFT
- история обнаружения периода

---
